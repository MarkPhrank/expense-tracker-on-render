💸 Expense Tracker (полностью в облаке)
Веб-приложение для учёта расходов с аналитикой, фильтрами и экспортом
🛠 Стек: Java 17, Spring Boot 3.2, PostgreSQL, Docker, Render, REST API, Vanilla JS
✅ Реализовал(а):  

    Многостадийную Docker-сборку с Maven, оптимизацию под 256 МБ ОЗУ  
    REST-контроллеры с CRUD, фильтрацией по датам/категориям, группировкой  
    Веб-интерфейс с экспортом в Excel (TSV) и печатью в PDF  
    Деплой на Render с переменными окружения и Internal Database URL
    🔗 expense-tracker-on-render.onrender.com
